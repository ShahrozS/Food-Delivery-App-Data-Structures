1- Install xampp ( given in the folder)
2- Start first two actions
3- open local host on your browser ( search local host) select phpmyadmin from top bar
5- click on new on left hand side to make a new database. name it fooddeliverytest1 (important)
6- now select that database
4- import the sql database given in the folder named fooddeiverytest1.sql
import option will be available on top bar 
5- run the project

if project fails to run you are missing sql connector.
